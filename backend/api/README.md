# FastAPI Backend API Layer

REST API for the Trust-Aware AI Learning Platform MVP.

## Overview

This API layer integrates all three engines and exposes them via REST endpoints:
- **AIProcessingLayer**: LLM queries with uncertainty tagging and confidence scoring
- **ChallengeEngine**: Verification question generation
- **SimplificationEngine**: Text simplification

## Endpoints

### POST /api/query
Query the LLM with uncertainty tagging and confidence scoring.

**Purpose:** Submit a learner query and receive an AI response with confidence scoring, uncertainty highlighting, and detailed confidence breakdown.

**Request Body:**
```json
{
  "query_text": "What is photosynthesis?"
}
```

**Request Fields:**
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `query_text` | string | Yes | The user's question or query (min 1 character) |

**Response (200 OK):**
```json
{
  "response_text": "Photosynthesis is the process by which plants convert light energy into chemical energy. [UNCERTAIN]The exact efficiency varies between 3-6% depending on conditions.[/UNCERTAIN] Plants use chlorophyll to capture sunlight and combine carbon dioxide and water to produce glucose and oxygen.",
  "confidence": {
    "score": 75.0,
    "justification": "The core explanation is well-established, but specific efficiency percentages may vary by source and measurement method.",
    "breakdown": {
      "question_clarity": "High",
      "topic_complexity": "Medium",
      "missing_information": "No"
    },
    "should_warn": false
  },
  "response_id": "resp_1709294400_abc123",
  "timestamp": "2026-03-01T10:30:00.000Z"
}
```

**Response Fields:**
| Field | Type | Description |
|-------|------|-------------|
| `response_text` | string | AI-generated response with optional `[UNCERTAIN]...[/UNCERTAIN]` tags |
| `confidence.score` | float | Confidence score from 0-100 |
| `confidence.justification` | string | Human-readable explanation of the confidence score |
| `confidence.breakdown.question_clarity` | string | "High" \| "Medium" \| "Low" |
| `confidence.breakdown.topic_complexity` | string | "Low" \| "Medium" \| "High" |
| `confidence.breakdown.missing_information` | string | "Yes" \| "No" |
| `confidence.should_warn` | boolean | True if confidence < 70 (warning threshold) |
| `response_id` | string | Unique identifier for this response |
| `timestamp` | string | ISO 8601 timestamp when response was generated |

**Error Responses:**
- `400 Bad Request`: Invalid input (empty query, validation error)
- `500 Internal Server Error`: AI processing failure or API key not configured

**Example with Low Confidence:**
```json
{
  "query_text": "What is the exact number of stars in the Andromeda galaxy?"
}
```

Response:
```json
{
  "response_text": "[UNCERTAIN]The Andromeda galaxy is estimated to contain approximately 1 trillion stars, though exact counts are impossible with current technology.[/UNCERTAIN]",
  "confidence": {
    "score": 45.0,
    "justification": "This question asks for a precise count that cannot be determined. Estimates vary widely in scientific literature.",
    "breakdown": {
      "question_clarity": "High",
      "topic_complexity": "High",
      "missing_information": "Yes"
    },
    "should_warn": true
  },
  "response_id": "resp_1709294500_xyz789",
  "timestamp": "2026-03-01T10:31:00.000Z"
}
```

### POST /api/challenge
Generate a verification challenge for an AI response.

**Purpose:** Generate a verification question that tests the learner's understanding of the AI response content.

**Request Body:**
```json
{
  "response_id": "resp_1709294400_abc123",
  "response_text": "Photosynthesis is the process by which plants convert light energy into chemical energy. Plants use chlorophyll to capture sunlight and combine carbon dioxide and water to produce glucose and oxygen.",
  "query_text": "What is photosynthesis?"
}
```

**Request Fields:**
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `response_id` | string | Yes | ID of the AI response to challenge (min 1 character) |
| `response_text` | string | Yes | The AI response text to generate a challenge for (min 1 character) |
| `query_text` | string | No | Original query text (defaults to empty string) |

**Response (200 OK):**
```json
{
  "challenge_id": "chal_1709294600_def456",
  "question_text": "According to the explanation, what are the two main inputs that plants use in photosynthesis?",
  "related_response_id": "resp_1709294400_abc123"
}
```

**Response Fields:**
| Field | Type | Description |
|-------|------|-------------|
| `challenge_id` | string | Unique identifier for this challenge |
| `question_text` | string | The verification question generated by the system |
| `related_response_id` | string | ID of the AI response this challenge relates to |

**Error Responses:**
- `400 Bad Request`: Invalid input (empty fields, validation error)
- `500 Internal Server Error`: Challenge generation failure or API key not configured

**Example Challenge Types:**

1. **Comprehension Check:**
```json
{
  "question_text": "What is the main product that plants create through photosynthesis?"
}
```

2. **Detail Verification:**
```json
{
  "question_text": "True or False: Photosynthesis requires both carbon dioxide and water as inputs."
}
```

3. **Concept Application:**
```json
{
  "question_text": "If a plant is kept in complete darkness, would photosynthesis still occur? Why or why not?"
}
```

### POST /api/simplify
Simplify complex text for easier comprehension.

**Purpose:** Convert complex or technical text into simpler language while preserving the core meaning.

**Request Body:**
```json
{
  "text": "Photosynthesis is the process by which plants convert light energy into chemical energy through a series of complex biochemical reactions involving chloroplasts and electron transport chains.",
  "response_id": "resp_1709294400_abc123"
}
```

**Request Fields:**
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `text` | string | Yes | Text to simplify (min 1 character) |
| `response_id` | string | Yes | ID of the response being simplified (min 1 character) |

**Response (200 OK):**
```json
{
  "original_text": "Photosynthesis is the process by which plants convert light energy into chemical energy through a series of complex biochemical reactions involving chloroplasts and electron transport chains.",
  "simplified_text": "Photosynthesis is how plants turn sunlight into food energy. Plants have special parts called chloroplasts that help them do this. They use sunlight to make the energy they need to grow.",
  "response_id": "resp_1709294400_abc123"
}
```

**Response Fields:**
| Field | Type | Description |
|-------|------|-------------|
| `original_text` | string | The original text that was simplified |
| `simplified_text` | string | Simplified version with easier vocabulary and shorter sentences |
| `response_id` | string | ID of the response being simplified |

**Error Responses:**
- `400 Bad Request`: Invalid input (empty fields, validation error)
- `500 Internal Server Error`: Simplification failure or API key not configured

**Simplification Strategies:**
- Replaces complex vocabulary with simpler synonyms
- Breaks long sentences into shorter ones
- Adds explanatory context for technical terms
- Maintains factual accuracy while improving readability

**Example with Technical Content:**

Request:
```json
{
  "text": "The mitochondria utilize oxidative phosphorylation to synthesize adenosine triphosphate molecules.",
  "response_id": "resp_1709294700_ghi789"
}
```

Response:
```json
{
  "original_text": "The mitochondria utilize oxidative phosphorylation to synthesize adenosine triphosphate molecules.",
  "simplified_text": "Mitochondria are parts of cells that make energy. They create a molecule called ATP, which cells use as fuel.",
  "response_id": "resp_1709294700_ghi789"
}
```

## Setup

1. **Install dependencies:**
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

2. **Configure environment:**
   ```bash
   cp .env.example .env
   # Edit .env and add your GROQ_API_KEY
   ```

3. **Run the server:**
   ```bash
   uvicorn backend.api.main:app --reload --port 8000
   ```

4. **Access the API:**
   - API: http://localhost:8000
   - Interactive docs: http://localhost:8000/docs
   - Alternative docs: http://localhost:8000/redoc

## CORS Configuration

CORS is enabled for:
- `http://localhost:5173` (Vite dev server)
- `http://127.0.0.1:5173`

All methods and headers are allowed for MVP simplicity.

## Error Handling

- **400 Bad Request**: Invalid input (empty text, invalid parameters)
- **500 Internal Server Error**: Processing errors (LLM failures, missing API key)

All errors return JSON:
```json
{
  "detail": "Error message here"
}
```

## Engine Initialization

All engines are initialized once at startup using the `lifespan` context manager:
- Reads `GROQ_API_KEY` from environment
- Initializes all three engines
- Logs initialization status
- Handles missing API key gracefully

## MVP Scope

This is an MVP implementation with:
- ✅ 3 core endpoints
- ✅ Basic error handling
- ✅ CORS enabled
- ✅ Engine integration
- ❌ NO authentication
- ❌ NO rate limiting
- ❌ NO advanced logging
- ❌ NO database persistence

## Architecture

```
FastAPI Application
├── Lifespan Management (startup/shutdown)
├── CORS Middleware
├── Request Models (Pydantic)
├── Response Models (from backend.models.schemas)
└── Endpoints
    ├── /api/query → AIProcessingLayer
    ├── /api/challenge → ChallengeEngine
    └── /api/simplify → SimplificationEngine
```

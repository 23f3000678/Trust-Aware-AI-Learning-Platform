# Trust-Aware AI Learning Platform - Frontend

React-based frontend for the Trust-Aware AI Learning Platform MVP.

## Features

- **Query Interface**: Submit questions to the AI
- **Confidence Display**: View confidence scores (0-100) with color coding
- **Confidence Breakdown**: See detailed breakdown of:
  - Question Clarity (High/Medium/Low)
  - Topic Complexity (Low/Medium/High)
  - Missing Information (Yes/No)
- **Uncertainty Highlighting**: [UNCERTAIN] text highlighted in yellow
- **Warning Banner**: Displayed when confidence < 70
- **Challenge Questions**: Generate verification questions (display only in MVP)
- **Simplification**: Toggle between original and simplified text

## Prerequisites

- Node.js 18+ and npm
- Backend API running on http://localhost:8000

## Installation

```bash
cd frontend
npm install
```

## Running the Development Server

```bash
npm run dev
```

The app will be available at http://localhost:5173

## Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## API Endpoints Used

- `POST /api/query` - Submit a query and get AI response with confidence
- `POST /api/challenge` - Generate a challenge question
- `POST /api/simplify` - Simplify response text

## Key Components

### QueryInterface
Main component that handles:
- User input form
- API calls to backend
- Response display with uncertainty highlighting
- Action buttons for challenge and simplification

### ConfidenceDisplay
Displays:
- Confidence score with color coding (green >70, red <70)
- Warning banner for low confidence
- Breakdown of confidence factors
- Justification text

### ChallengeDisplay
Shows challenge questions generated by the backend. MVP version displays only (no answer evaluation).

### SimplificationToggle
Allows toggling between original and simplified versions of the response text.

## Styling

Basic CSS with:
- Responsive design
- Color-coded confidence indicators
- Yellow highlighting for uncertain text
- Clean, modern UI

## MVP Limitations

- No user authentication
- No routing (single page)
- No complex state management
- Challenge questions display only (no answer evaluation)
- Basic error handling
- No persistent storage

## Development Notes

- Built with React 18 + TypeScript
- Vite for fast development and building
- Axios for API calls
- No external UI frameworks (pure CSS)
- Types match backend Pydantic models
